# Playground 布局指南

## 新布局概览

JSON Visual Diff Playground 采用了全新的左右分栏布局，优化了用户体验。

```

│  Toolbar: [示例选择] [导入] [导出] [设置]                        │
└─────────────────────────────────────────────────────────────────┘
┌──────────────────────────────┬──────────────────────────────────┐
│                              │                                  │
│   左侧编辑器面板 (50%)        │   右侧结果面板 (50%)             │
│                              │                                  │
│  ┌────────────────────────┐  │  ┌────────────────────────────┐ │
│  │  原始 JSON (Old)       │  │  │  差异结果                  │ │
```

## 区域说明

### 1. Header（顶部标题栏）
- 显示应用名称和描述
- 固定在顶部，不随页面滚动

### 2. Toolbar（工具栏）
- **示例选择器**：快速加载预设示例
- **导入按钮**：导入 JSON 文件到左侧或右侧编辑器
- **导出按钮**：导出差异结果为 JSON 或 HTML
- **设置按钮**：打开编辑器设置面板

### 3. 左侧编辑器面板（50% 宽度）

#### 原始 JSON 编辑器（上方）
- 输入要比较的原始 JSON
- 完整的 Monaco Editor 功能
- 实时语法验证

#### 新 JSON 编辑器（下方）
- 输入要比较的新 JSON
- 完整的 Monaco Editor 功能
- 实时语法验证

#### 比较按钮（底部）
- 点击执行 JSON 比较
- 结果显示在右侧面板

### 4. 右侧结果面板（50% 宽度）

#### 结果头部
- 显示"差异结果"标题
- 显示统计信息（新增、删除、修改、未变）

#### 结果内容
- 可视化显示 JSON 差异
- 支持展开/折叠
- 颜色编码：
  - 🟢 绿色：新增的字段
  - 🔴 红色：删除的字段
  - 🟡 黄色：修改的字段
  - ⚪ 灰色：未变化的字段

## 布局优势

### 1. 垂直编辑器排列
- ✅ 更适合查看长 JSON 文档
- ✅ 减少水平滚动
- ✅ 更符合阅读习惯

### 2. 左右分栏
- ✅ 同时查看输入和输出
- ✅ 充分利用宽屏显示器
- ✅ 减少上下滚动

### 3. 50/50 分屏
- ✅ 平衡的空间分配
- ✅ 编辑和查看结果同等重要
- ✅ 视觉上更和谐

### 4. 响应式设计
- ✅ 自动适配不同屏幕尺寸
- ✅ 移动端友好
- ✅ 触摸操作优化

## 使用建议

### 桌面端使用
1. 使用宽屏显示器获得最佳体验
2. 左侧编辑，右侧实时查看结果
3. 使用代码折叠管理大型 JSON

### 平板端使用
1. 横屏使用以获得更多空间
2. 上下滚动查看编辑器和结果
3. 使用触摸手势操作

### 手机端使用
1. 竖屏使用，垂直滚动
2. 先编辑 JSON，再查看结果
3. 使用代码折叠减少滚动距离

## 快捷操作

### 编辑器操作
- 双击：选择单词
- 三击：选择整行
- Ctrl+滚轮：缩放字体
- 拖拽：选择多行

### 结果面板操作
- 点击折叠图标：展开/折叠节点
- 滚动：查看完整结果
- 复制：选择文本后复制

## 自定义布局

虽然当前布局是固定的，但你可以通过以下方式自定义：

1. **调整浏览器窗口**：改变整体大小
2. **使用浏览器缩放**：Ctrl+/- 调整缩放级别
3. **编辑器设置**：调整字体大小和主题
4. **代码折叠**：隐藏不需要的部分

## 常见问题

### Q: 可以调整左右面板的宽度吗？
A: 当前版本是固定的 50/50 分屏，未来版本可能会添加可调整的分隔条。

### Q: 移动端可以横屏使用吗？
A: 可以！横屏时会提供更多的水平空间，但布局仍然是垂直堆叠的。

### Q: 编辑器太小怎么办？
A: 可以：
1. 最大化浏览器窗口
2. 使用全屏模式（F11）
3. 调整浏览器缩放级别
4. 使用代码折叠

### Q: 结果面板可以单独打开吗？
A: 可以使用"导出 HTML"功能，在新窗口中打开结果。
